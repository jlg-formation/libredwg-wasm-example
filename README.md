# LibreDWG with WASM

## Install

Prerequisites:

- OS: Linux or Mac or Linux under Windows (WSL)
- GNU build essentials (gcc, make, etc.)
- Emscripten (https://emscripten.org/)
- curl

```
git clone https://github.com/jlg-formation/libredwg-wasm-example.git
make
```

## Use

```
make test
```
